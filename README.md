# Nokia-Snake-Game

Nokia Snake Game

Description

Nokia Snake Game is a browser-based recreation of the classic Snake game found on early Nokia mobile phones. Built using HTML, CSS, and JavaScript, the game features smooth controls, score tracking, collision detection, and a retro-inspired interface.

This project is ideal for beginners who want to learn JavaScript game development, HTML5 Canvas, and DOM manipulation.

⸻

Features

* Classic Snake gameplay
* Retro Nokia-inspired interface
* Keyboard controls
* On-screen controls for mobile devices
* Real-time score counter
* Food generation at random locations
* Collision detection with walls and the snake’s body
* Game Over screen
* Responsive and lightweight design

⸻

Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* HTML5 Canvas

⸻

Installation

1. Clone the repository:

git clone https://github.com/yourusername/nokia-snake-game.git

2. Navigate to the project folder:

cd nokia-snake-game

3. Open index.html in your preferred web browser.

No additional installation or dependencies are required.

⸻

How to Play

* Press any arrow key to start moving the snake.
* Eat the food to increase your score.
* Avoid hitting the walls.
* Avoid colliding with the snake’s own body.
* The game ends when a collision occurs.

Controls

Key	Action
↑	Move Up
↓	Move Down
←	Move Left
→	Move Right

Mobile users can also use the on-screen directional buttons.
